# Mighty Andventures of the Regular Everyday Normal Dude!
Jogo RPG controlado por texto, desenvolvido em dupla para a disciplina de Programação Orientada a Objetos na linguagem Java.

Para executar o jogo pode-se compilá-lo através dos arquivos .java da pasta src ou executar o arquivo MAREND.jar da pasta dist.

O jogo é controlado por comando de textos como:
go back
pick can
attack skeleton
